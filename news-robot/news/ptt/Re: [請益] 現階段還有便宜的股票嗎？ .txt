: 你可能需要自已先定義何謂便宜的股票，便宜這件事很多人的定義都不同，殖利率、pe、
: 十年線等等，有很多種判斷便宜股票的指標，有些人用1.2種指標判斷，有些用了很多指
標
: 去綜合判斷，但不管何種方式，都並不一定適合自已。
: 而且股票大多數的時間都處在二種狀況間遊走。
: 第一種狀況：股票現在看起來不便宜，但評估未來的營運狀況後，可能是便宜的。
: 第二種狀況：股票現在看起來很便宜，但評估未來的營運狀況後，一點也不便宜。
: 拿景氣循環股來講可能會比較好理解，景氣不好的時候，本益比20.30倍甚至幾十倍都有
，
: 但未來景氣回復的時候，現時的股價反而是便宜的。
: 如果在景氣好的時候，本益比可能也才10倍，甚至不到10倍都有可能，看起來好像很便宜
: ，但實際上這時候進場，是很有可能套牢。
: 因為股市常態上是處在於上述二種狀況下遊走，所以要找便宜的股票，就要去評估這二種
: 狀況。
: 除非你所謂的便宜股票是要在股災下，不管好股票或爛股票都跌成屎，但通常在這個狀況
: 下，你也不一定敢買，那時候低還有更低，要一直接下去，可不是一般人承受的住的。
: 就算你敢買，除非你是老巴，不然你也沒有無限子彈一路接下去。
: 就以我來說，我在去年約14000點去借信貸來買，結果一路買到反彈15000，又一路往下買
: ，但還沒買到大盤落底12629，我的子彈就打光了，所以我只能看著乾瞪眼，等著慢慢漲
回
: 來，又或者漲不回來，其實我當下我也不知道，不過我還是對我買的股票有信心就是了。
: 但現實是落底後，立馬反彈，快到根本就沒什麼時間反應。
: 所以如果你一直想等股災到來，才是便宜的股票，這個時候才要進場，有很大的概率是你
: 大概這輩子也不會買到股票。
: 或許因為這樣，老巴才會說，他情願在不知道股價是多少的情況下，去評估一間公司的價
: 值，這樣才不會被市場的價格影響自已的公正客觀。
: 像我自已在很多時候，也是在初步研究完公司的狀況後，才會知道現在的股價有沒有在合
: 理的範圍內，有沒有讓我可以更深入研究的價值。但在很多時候，初步看很便宜，但深入
: 研究後，覺得不便宜。反而一些在我看來不是便宜，但合理的價格，深入研究後，發現很
: 有長期持有的價值。
: 寫了這麼長，其實結論很簡單，不要一直想要買在最低，也不要想說等股災才要進場，因
: 為永遠沒辦法知道那邊是最低點，更多的情況下，是只能作出當下最好的判斷而已。
