這是一套書 ( 祖魯法則 ; 超越祖魯法則 )
序 : 大象不會急奔 ( 聚焦一個展望非常好的領域或類股 )
簡單來說 : 專注特定領域 ( 非漫無目的亂買 )
作者做法 : 聚焦小型股或微型股
作者選股方法 : (選股最基本條件)
1.      股利至少4 %
2.      股本盈利過去5年內，至少有4年成長。
● 過去5年每股盈餘複合成長率大約為15%
本益成長比 : 低於0.75 甚至低於 0.66 (主要篩選出價值股)
本益成長比公式 : (PEG) 大家可以去看市場先生的介紹
https://rich01.com/ratio-peg/
3.      股本盈利過去4年間至少必須倍增。
4.      董事長最新的聲明必須樂觀
5.      公司必須擁有合理流動性
流動性高、高現金流、低債務
6.      公司必須不受特殊因素傷害
競爭優勢 : 可口可樂/健力士 (本身強大品牌)
我看來就跟過去很多網紅財經專家說的企業有護城河
題材性 : 電動車、ESG、疫苗類…
7.      股票必須擁有合理的資產價值 : 注意公司淨值
8.      公司不應該由家族控制 :
經營管理層持股需高 (非家族企業管控)
9.      股票應該有投票權
