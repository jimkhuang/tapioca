根據一些已經公布的消息與新聞

首先.台積3奈米良率85% 應該是基於這篇,開幕上董事長說N3良率與N5初期相當

https://pr.tsmc.com/chinese/news/2986

然而業內人士都知:

台積 3 奈米傳「良率近八成」大贏三星？外媒：兩者無法這樣比較

https://technews.tw/2023/01/03/tsmc-3nm-4/

為什麼會這樣說,因為每一種晶片大小不一,真的要評論. 其實要用D0

那麼台積的D0 是多少呢?

https://reurl.cc/aaK1DD

早在2020台積技術論壇時有公布N5 在MP ( mass production)的D0 約在1.1左右

由於下線到出貨約莫1Q, 因此MP-1Q 時的D0約在0.14左右

再根據劉董事長在 N3量產典禮上的說詞 N3 = N5 推測目前台積D0 約莫0.14


https://zh-tw.pdf.com/applications/yddc.html

根據D0 及die size 換算yield 的話.

以N5 最初量產的晶片A14來看, die size 為1.14cm^2

可以得到 yield 約莫85.2%.  與報導相符.

( 報導根據 N3 = N5 初期 = 85% 來推測 N3 良率85%)

然而N3 第一顆量產的產品, 推測並非A17, 而是M2 MAX

https://ctee.com.tw/news/tech/741663.html

由於M2 PRO 尚未上市,無法知道 DIE SIZE

但由於APPLE 歷代晶片面積差異不大,因此先套用M1 MAX的晶片SIZE

D0 = 0.14, die size = 4.25  可以推算出yield 約55.2%

我想這就是這篇傳言的 良率約莫50%


接著我們回到3星

早在去年初就有 三星與高通 包莖萬 良率只有30% 的新聞

而包莖萬的die size 約莫1.9cm^2 可以知道三星 4nm 的D0 約莫0.6

由於三星幾乎沒有在任何場合公開過自己的D0

(tsmc 可是很大方公開的)

先假設3nm D0 有如tsmc 對標4nm 好了

另外眾所周知,三星3nm早期客戶只有一間名不見經傳的挖礦晶片

當然這個晶片的die size 也無從得知.

(以上兩個D0 /  die size,有知情人士希望能透露一下)

所以先從挖礦晶片最有名的bitmain 彼特大陸5nm晶片來推敲

彼特大陸5nm晶片的die size 約莫 0.12cm^2

由此推算會得到93%的良率.  因此報導所稱良率趨近完美就出來了


當然D0 0.14 V.S 0.6  誰贏誰輸大家自行猜測吧.


PS1. 如果以一般手機晶片1.0cm^2 +之前三星3NM 良率只有10%推算

     三星3NM D0 可能實際是1.0 而非0.6. 就算套用D0=1.0 die size =0.12

     還是可以有88%的良率因此與報導依然不衝突

PS2. 報導說良率完美是"三星" 高層所述, 而非"三星電子"高層所述

     大家都知道三星少主李在榕 是個 歷史系+ MBA. 簡單的說就是文科仔

     D0 甚麼的我猜他也聽不懂啦.如果三星高層哪麼厲害,當初會有這新聞?
