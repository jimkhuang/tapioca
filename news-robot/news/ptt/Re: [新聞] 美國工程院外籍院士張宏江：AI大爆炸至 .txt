: 工智慧（AI）未來會超越人類智慧嗎？如今 AI 行業在這個問題上已基本達成共識。
: 北京智源研究院給出了通用人工智慧（AGI）的釋義：一定是比人類強大的智慧，才真正
: 叫 AGI。所以，自主智慧、超人智慧、強人工智慧，其實講的都是一種全面超越人類的智
: 慧。
: 過去三年來，《深網》與智源研究院理事長、美國國家工程院外籍院士張宏江多次對話。
: 他做過研發，做過管理，也做過投資，外界熟知的身份是金山集團與金山雲執行長、微軟
: 亞洲研究院共同創始人。以下是《深網》與張宏江的對話摘要。
: Q：您認為我們現在處於 AI 技術發展的哪一個階段？
: A：AI 過去的十年是積累的階段，今天會看到一個大爆炸的時點。這個大爆炸實際上是在
: 深入到每一個可能的應用場景裡面去。這也意味著 AI 的門檻在降低，或者說 AI 本身變
: 成一個基本的技術能力。
: 所以我也不認為大學裡設人工智慧系是件特別好的事。它已經成為一個基本技能的話，電
: 腦科學系的學生都要學啊。就像你今天在電腦科學系要學系統架構或者學數據庫，你不會
: 再設一個數據庫系，或者系統架構系。
: Q：大爆炸期會持續多久？
: A：最少十年。
: Q：李開復不支持將 AI 比喻為類人腦，認為人類對人腦的了解太少，您認同嗎？
: A：我們不應該把今天的人工智慧、深度學習說成是類人腦，或者是像人腦一樣思考，確
: 實不是。今天的 AI 不是人腦這種思維方式的一個很重要的原因，是我們對人腦的認識很
: 少，所以我們更應該去學習人腦。
: Q：未來的 AI 發展將會給人類社會帶來怎樣的根本性變化？
: A：Yuval Noah Harari 寫過一本書，叫《未來簡史》，他想強調的是技術的發展會導致
: 人類的分化，一些人會永遠落在技術後面，成為所謂的弱者，或者說閒人。
: 他這個說法可能有點誇張，但我覺得我們是時候來考慮這個問題了。當技術發展的速度超
: 過人的學習能力的時候，他的工作會被機器所取代，如果不轉變就不可能再找到另外的工
: 作。
: Q：是必然會發生的嗎？
: A：必然的。從工業革命到現在資訊革命，新技術滿足的精神需求越來越多。Facebook 不
: 生產任何衣服，也不生產任何糧食，但他的公司營收比任何一家做衣服或者做糧食的公司
: 都要多。
: 技術所帶來的兩極分化，可能最終會導致經濟上的兩極分化變得越來越厲害。
: 我們現在想想，為什麼這麼多人反對全球化。完全的全球化意味著這個世界上這一份工作
: 只要在另外一個地方比你這個地方便宜，這份工作就會在這消失。按這個思路想，機器人
: 時代這份工作只要有個機器人能做，這個工作就會消失。
: 但一方面，我們不可能停止或者拖慢技術的發展，另一方面，人們的適應和學習能力也不
: 太可能加速。這個矛盾未來怎麼解決？這種挑戰用傳統的方式可能解決不了。資訊技術出
: 現後，新產生的工作比他消滅的工作要多，技術發展能夠自己把這個問題解決掉。
: Q：技術發展會帶來世界秩序的重建嗎？
: A：當大部分變成閒人的時候，社會秩序必須得重建。這種秩序未必是國家的變化，但經
: 濟體系是一定要變的。
