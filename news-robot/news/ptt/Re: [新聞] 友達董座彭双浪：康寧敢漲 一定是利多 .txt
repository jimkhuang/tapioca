: 璃基板將調漲售價2成，在面板產業仍處於虧損情況下，康寧喊漲到底是利多還是利空，
: 市場看法分歧，不過友達（2409）董事長彭双浪昨表示，「能喊漲價一定是利多」，康寧
: 為何敢漲價？就是看到面板需求往健康方向走。
: 庫存決定價格 康寧敢漲有理
: 彭双浪說，過去兩年所有產業最流行的一個名詞就是「庫存」，而客戶清庫存動作對友達
: 影響很大，雖然經濟學教科書教我們產品價格是由市場供需決定，實際上，庫存才是真正
: 決定產品價格的關鍵。
: 庫存高的時候，大家減少庫存，產品價格大幅滑落；庫存低到一個程度，大家開始拉貨，
: 價格就上升。「康寧為何敢漲價，也是因為它看到面板供需平衡反應比其他產業早很多」
: 。
: 彭双浪說，友達客戶從2021年第四季就已經在調整庫存，迄今已調整了7個季度，電視面
: 板是第一個被下調的產品，也是第一個上來，現在PC相關產品也看到急單，有急單就意味
: 庫存回到健康水位甚至是低於平常水準。
: 「高速公路塞車 得前面車先動」
: 當然每個產業受庫存影響程度不同，復甦速度也不一致，彭双浪以「高速公路塞車理論」
: 比喻。在高速公路上，當前面出現路況，第一輛車停下來，後面的車還在以每小時100公
: 里的速度前進，等到前面故障排除了，開始啟動往前跑，後面的車子至少還要10分鐘才能
: 啟動。
: 友達昨日舉行股東會，小股東也關切友達去年承諾的3年股東報酬計畫能否兌現？彭双浪
: 說，「我們當時告訴股東3年股利會達到6.5元，扣掉去年已發放3元，還有3.5元，考量去
: 年虧損，加上後面還有很多轉型需求，在營運資金與股東報酬平衡下，今年以資本公積發
: 放現金股利0.8元，剩下部分還在努力，希望未來能夠達成目標。」
