: 針對迪士尼點評
: Bob Chapek是公園部門上來的新CEO 他的擅長領域就是裁員跟節省成本 (Cost-down)
: 當年BoB Iger在還沒退位前 公司內部都紛紛猜測下一個接班人應該會從Disney Plus
: 上來 然而在併購完福斯影視之後 Bob Iger卻把王位交給公園部門上來的BoB Chapek
: 導致Disney Plus的上層原以為會得到皇位的人 紛紛出走去其他公司
: 現在回過頭來 這一切都很明顯 在併購福斯之後 迪士尼的現金低到史上新低
: 因此勢必需要一位新CEO 做開源動作 當2019 BoB Iger把皇位交給BoB Chapek時
: 這已經預告接下來的迪士尼 就是開源節流(裁員) 順帶一提 我個人認為併購
: 福斯為了拿下Hulu另外30%股權 現在事後看來是一個錯誤的決定 他造成迪士尼手上現金
: 狂燒 在2018電影這樣奇蹟大賣 卻因為併購福斯 以致無法有足夠現金因應任何災難
: 然後在2020剛好遇到疫情大爆炸  使得迪士尼只能走無限cost-down去維持現金流
: 因此2020開始 迪士尼就陸續關店北美Disney Store 亞洲這部分關掉許多部門
: 甚至最近這期把Glendale/Burbank辦公室的人 強迫搬去佛羅里達為了省下薪水
: (這提案也是公園部門Josh提出 已經引起許多人不滿離職) 造成許多優秀人才出走到
: 其他公司像是AirBnB  Disney目前對外招員工都是以Project Hire 為主(迪士尼把約聘
: 稱為Project Hire 弄的一個好聽名詞)
: 因此這幾年迪士尼會不斷走向省錢 高工時 低薪水 連最近的黑寡婦事件 都一開始
: 高姿態不想給分紅 因此今天當所有好萊屋電視電影工會第一次128年 以98%
: 高票贊成罷工時 我個人覺得這是迪士尼接下來的最大挑戰 當罷工如果成功後
: 就意味者以後製造Content成本會上升
: 針對Netflix點評
: 就在其他影音平台都在搶美國的電影Content Netflix卻走向世界 開始用亞洲的有才華
: 導演和內容 來打世界觀眾 相較迪士尼侷限在深耕美國題材 但面臨手上現金不夠
: 因此我個人非常看壞迪士尼和看好Netflix
: 最後追加一個其他消息當參考 迪士尼的研究員一年薪水大概是15萬美金 對比Netflix
: 一年薪水是45萬美金現金 我覺得迪士尼遲早有一天會陷入低薪找不到好人才情況
: 我的預測是 Netflix會從日本卡通漫畫 搭配韓國電影和電視劇 來回頭跟好萊屋對打
: 如果好萊屋這次電視電影接下來工會罷工成功 那也許我們會看到更多影音串流平台
: 來亞洲或者歐洲找題材 而不在以美國看天下

: 應該有機會改變電影產業的佈局
