原文內容：

外媒報導，英特爾下一代 Arrow Lake CPU 是首款採用 Intel 20A 打造的 CPU，但計畫
生變，英特爾放棄 Intel 20A 製程，轉向採用台積電 3 奈米打造 Arrow Lake CPU。


市場傳聞指出，英特爾可能放棄 Intel 20A 打造 Arrow Lake CPU，轉向台積電 3 奈米
。台積電 3 奈米製程的 N3B 是 Arrow Lake CPU 的潛在候選製程。

台積電 3 奈米製程系列有 N3E、N3P、N3X 等，性能更高和功耗更低。Intel 20A 本有望
是台積電 3 奈米的強大對手，但看起來 Arrow Lake CPU 可能會放棄。另一傳聞指出，
英特爾最新製程藍圖不再列出 Intel 20A，因已轉向外部代工廠生產 Arrow Lake CPU。

英特爾 CPU 採小晶片設計，CPU 晶片採 Intel 20A 製程，GPU 晶片採台積電 3 奈米，
IO 和 SOC 晶片也採台積電 3 奈米製程。不過傳聞 Arrow Lake CPU 將採用台積電 3 奈
米打造所有整合其中的小晶片。

英特爾已跳過 Meteor Lake-S 桌上型 CPU 轉用 Arrow Lake-S CPU，2024 年第四季或
2025 年第一季推出。英特爾還將用 Arrow Lake 第二代 Core Ultra 系列取代行動產品
系列，尚未確認這些改動是否僅適用特定市場或所有市場。

目前英特爾還未確認此傳聞，外界都非常注意英特爾如何回覆。資訊來源曾有非常準確的
紀錄，故 Intel 20A 很可能真的又延後或取消，英特爾不得不靠台積電先進製程生產下
一代 CPU 主要製程。
