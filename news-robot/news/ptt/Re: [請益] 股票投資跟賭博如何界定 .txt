: 可能是起源於大航海時代(大家集資去湊船  船能夠發大財就大家分紅  反之血本無歸)

: 接下來的描述算我個人主觀的意見  就看看就好
: 1:所謂的投資
: 在我個人的認知中
: 看好某產業的後續發展(實際營收有辦法提高)/持續獲利擴大
: 不過多數散戶都是會追高  等到套牢之後才用:我是價值投資來騙自己
: 如果真的是所謂的價值投資
: 那就是回歸到計算財報/本益比是否過高之類的
: 例如本益比10倍的股票  投入的本金
: 公司10年就可以翻倍
: 本益比50被的股票  投入的本金
: 需要50年才可以翻倍
: 但是多數人的投資都是往高本益比的股票衝進去(最近的ai股之類的)
: 投資不只是說一直買好公司的股票
: 還要選擇在這公司的股票相對低的時候在開始購買
: 才真的算是所謂的價值投資
: 很多人都是用價值投資在說服自己  XD
: 本身是走瘋狗交易模式(就是看到狂漲的就追進去  追高不是問題  但是該跑要跑)
: 但是該停損的時後又轉為價值投資模式(實際上是被套牢)
: 巴菲特我覺得算是投資的神(投資類/不是投機類)
: 因為他都會挑優質公司股價相對低的時候才進場購入(不會隨意追高)
: 舉例來說
: 一支股票100到400(近期的nvda)  漲了四倍
: 業績一定是有所成長(但是真的有成長到四倍嗎)
: 真的所謂的價值投資  應該是根據該公司的業績之後自己去做功課計算
: 等到真的跌深之後再找機會出手(通常要等幾個月/甚至幾年)
: 2:投機
: 目前絕大多數的交易多是投機
: A類型找個話題=>A類型股票亂噴==>散戶追進去==>山頂洗碗==>B話題(重複循環)
: 投資可不可以賺錢  當然可以  但是就回歸到交易重點
: 該停損就是要停損
: 就像投資一樣  如果原先看好的公司走下坡  難道還是資金永遠留在裡面嗎?
: Nokia以前是手機王者
: 後來智慧手機出來之後直接趴下(第一名也可以因為趨勢變化下台)
: 該減碼就是應該減碼
: 當然我個人研究的技術分析其實也就是投機
: 不過黑貓白貓抓地到老鼠都是好貓
: 投資投機都是要花時間研究的
: 3:賭博
: 完全不做任何分析(分析不準應該是改進自己的分析方式)
: 就是憑話題/感覺直接進行交易
: 就算是德州撲克這種賭博
: 也有人可以拿世界冠軍(也是有打牌的一些技巧/分析)
: 想賺錢就是要花時間去研究/還有繳學費
: 能提高賺錢機率的方式通常都要花錢去學(甚至還可能學到假的  XD)
: 但是股票交易的重點我覺得就是有歷史資料可以查詢
: 自己可以根據交易邏輯去回測就知道是否有用(或調整)
: 最後還是要強調  別隨便無腦多  因為16200跌破沒站回去  後續可能真的殺很大
