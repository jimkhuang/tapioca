基本面沒用是大部份的人都只會看過去的基本面

真正的基本面是要預測產業趨勢,企業成長動力,風險因子分析

去抓出未來半年到一年之後的企業的基本面變化

舉例說一間公司去年的EPS 6 目前股價 120 你會說本益比高達20

在相同產業族群裡面是高標,所以目前股價算是合理價,短期不宜追高

但懂得預測分析基本面的會看出,今年此公司業績成長有機會20%

帶動EPS今年向上成長30%-40%,等於上看8

換算成目前的股價本益比才15倍,相較於目標本益比18-20倍還有20%-30%的獲利範圍

所以會在這個股價附近大力買進

這才是真的靠基本面吃飯的人的思維,而不是看已經發生財報去推算合理價

那才叫做是看後視鏡開車

: : 看股票不外乎三種麵

: : 籌碼就是三大法人融資券數量
: : 基本面就是財報
: : 其中基本面最沒用，知道之後通常都已經漲完跌完了
: : 但技術面與籌碼面
: : 是天天在更新

: : 當技術面呈現支撐、低檔起漲多頭排列

: : 這種
: : 這樣要怎麼判斷呢？
: : 哪個會比較真呢？
: 這三個面小弟認為其實是三個不同時間長度的影響
: 基本面是長期（季-年）
: 籌碼面是中期（週-月）
: 技術面是短期（日-週）
: 基本面（產業趨勢、產品報價、市場供需等）是影響公司能否獲利向上的原動力，我覺得像
: 是汽車的燃油是否加滿，能否走的長遠
: 籌碼面（法人買賣、大戶增減、融資融券）是影響股價是否上漲的原因，我覺得像是汽車的
: 引擎是否發動熱機，能否改變盤整趨勢
: 技術面（均線、量價、技術指標）是最後的結果，基本上都是股價漲了，上述的指標均線才
: 跟著變化，我覺得像是油門，催了之後才會看到車子已經向前衝了
: 你說基本面沒用應該是指在短線上，股價是否上漲會和基本面會脫鉤
: 所以你的目標應該是想知道股價起漲，到底是漲真的還是漲假的
: 籌碼大亂的定義我不太懂
: 如果是指單日融資大增，或是單日法人大賣
: 因為你不知道融資是散戶還是主力，你也不知道外資是假外資隔日沖還是真外資
: 所以一兩天的籌碼變化其實沒有太大的意義，除非你知道那些籌碼是誰買賣的
: 籌碼的變化影響其實要看個幾天才有比較明顯的改變
: 重點是該籌碼的變化能否影響股價的走勢
: 通常大股東的變化比較有影響力，因為大股東是有錢人，而股價是用錢堆出來的
: 回到你的問題，籌碼面、技術面要看哪個？
: 我覺得如果你是順勢交易買一個波段的，要以技術面為主
: 看到車子（股價）往前過了停止線（均線），那就應該在第一時間上車
: 引擎有沒有熱好（籌碼有沒有集中），只是輔助你猜測車子能否爆發衝刺（股價大漲）
: 不代表引擎沒熱（籌碼散亂），車子就不會前進
: 最後舉個例子6669的緯穎
: 可以看到籌碼面千張大戶連續減少
: 股東人數暴增（籌碼散亂不集中）
: 結果股價噴上天
: 如果只看籌碼面就會錯過這波大漲
: 你說要以哪個面為主呢？
