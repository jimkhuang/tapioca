記者蕭文康／台北報導

宏碁在COMPUTEX前宣布推出包括全新Predator Triton 16（PT16-51）電競筆電、Acer
Swift Edge 16（SFE16-43）筆電及Acer Connect Vero W6m網狀路由器，積極搶攻展場題
材商機。

宏碁指出，全新Predator Triton 16（PT16-51），為效能強大的電競筆電系列再添生力
軍，適合用於遊戲和工作。Predator Triton 16搭載最新第13代Intel Core™處理器、
NVIDIA GeForce RTX 4070筆電GPU 、最高配置32 GB的DDR5 RAM記憶體以及豐富的功能，
使各類型的玩家感到熱血沸騰。

Predator Triton 16進階的散熱解決方案將兩個第5代 AeroBlade™ 3D 金屬風扇、Acer
Vortex Flow技術和CPU上的液態金屬散熱膏，整合至纖薄的金屬機身設計中，確保重要零
組件在遊戲激戰下，隨時保持通風涼爽。

Predator Triton 16具備時尚外觀和強大效能，厚度僅1.99 公分，機身採用精緻、且閃
亮耀眼的銀色金屬外殼，使玩家在外出移動時，不僅輕鬆便攜還能感到時尚自信。全新的
輕薄電競筆電擁有強大運算能力，配備第13 代Intel Core i9處理器，支援最高Turbo時
脈5.4 GHz 的全新效能混合架構，其中的CPU與NVIDIA® GeForce RTX™ 4070筆電GPU、
NVIDIA DLSS 以及Max-Q技術功能相互協作，全面優化光線追蹤和整體性能。Predator
Triton 16可配置高達32 GB DDR5 5200 MHz記憶體和高達2 TB PCIe M.2 SSD，採用RAID
0配置，提供玩家強大的效能體驗。

Predator Triton 16附贈一個月的Xbox Game Pass Ultimate。玩家可透過優異的顯示設
計體驗身臨其境的視覺饗宴，Predator Triton 16採16吋（16:10）WQXGA 2560 x 1600
IPS顯示器，支援500尼特的峰值亮度、超高240 Hz螢幕刷新率和窄邊框設計，同時具備
100% DCI-P3 廣色域與NVIDIA® Advanced Optimus和NVIDIA® G-SYNC®技術，能夠在畫
面精細、快速移動的遊戲場景中，呈現流暢無阻、栩栩如生的視覺效果。

為了讓玩家隨時保持備戰狀態，Predator Triton 16擁有頂尖的散熱技術，採用雙風扇系
統和專門設計的第5代AeroBlade 3D金屬風扇，並藉由Acer Vortex Flow技術，進一步突
破散熱技術的極限，透過氣流導引結構與出風口保持內部涼爽，搭配CPU上的液態金屬散
熱膏，使熱傳導與效能更上層樓，確保筆電以最佳效能運行。

玩家可透過PredatorSense按鍵控制遊戲設置與自訂偏好，包含每個鍵盤按鍵的RGB背光顏
色、使用者介面、管理控制系統以及 PredatorSense專用應用程式上的多種設定。
Predator Triton 16支援Windows 11系統，配備Intel Killer DoubleShot Pro （
E2600+ Wi-Fi 6E 1675i）及大量功能的連接埠，包括兩個USB 3.2 Gen 2連接埠、HDMI
2.1、一個附Thunderbolt 4 的 Type-C USB 連接埠和一個 Micro SD讀卡機，同步支援
Windows Hello 的指紋辨識器，提供更多防護與解鎖方式。

▲Acer Swift Edge 16筆電。（圖／Acer提供）

另外，宏碁全新升級的Acer Swift Edge 16（SFE16-43）筆電，專為需要強大運算能力、
高效生產力和便攜性的專業行動工作者所設計。此款輕薄筆電搭載AMD Ryzen 7040系列處
理器，最高可配備AMD Radeon 780M顯示晶片，提供次世代的性能和視覺效果，並在AI發
展浪潮下領先群雄，能於指定處理器型號上配備AMD Ryzen AI。

Acer Swift Edge 16具備令人驚豔的16吋3.2K OLED顯示器，螢幕刷新率升級至120 Hz，
並支援100% DCI-P3色域，能提供最生動、逼真的色彩和影像效果；並搭載Windows 11，
最高能支援Wi-Fi 7[1]連線，更透過Microsoft Pluton安全技術和一系列智慧功能擴大相
容性，讓使用者在享有高速無線網路時，也能確保隱私資料的安全。

Acer Swift Edge 16擁有使人的吸睛的外觀設計與出眾的色彩顯示能力，使人眼睛為之一
亮。超輕薄鎂鋁合金機身厚度僅1.295公分，重量僅1.23公斤，採用有光澤的橄欖黑塗料
。Acer Swift Edge 16的16吋3.2K OLED（3,200 x 2,000）顯示器支援120 Hz螢幕刷新率
、100% DCI-P3色域，及0.2毫秒反應時間，讓使用者得以盡情享受劇院級的視覺體驗，並
結合1,000,000：1對比度，500尼特的峰值亮度，呈現精準且高對比的清晰色彩，更通過
VESA DisplayHDR True Black 500認證，且為獲德國萊因TÜV顯示認證的顯示器，有助減
少藍光，提高長時間觀看的舒適度。

Acer Swift Edge 16透過最新AMD Ryzen 7040 系列處理器[1]，提供優異的效能速度和耗
電效率，在不插電的情況下能擁有更長的續航力，而最高搭載AMD Radeon 780M的顯示晶
片能為使用者帶來沉浸式的視覺饗宴；配備特定處理器的機款則採用 AMD Ryzen AI技術
，為使用者打造嶄新的AI使用體驗，包括視訊通話的即時視訊品質升級。透過內建預設的
Microsoft Pluton安全處理器，Swift Edge 16能提供抵禦惡意複雜攻擊的額外保護，而
Windows Hello的指紋辨識器能執行生物辨識驗證，讓使用者輕鬆登入採用Windows 11作
業系統的筆電。

Acer Swift Edge 16配備高達32 GB的LPDDR5 記憶體和高達2 TB的PCIe Gen 4 SSD儲存空
間，可實現高速資料傳輸、更低的延遲和更短的載入時間，而TwinAir冷卻系統配有改良
型風扇，加上鍵盤進氣口的精心設計，能有助於維持高效的全速運行。

最新版本的Swift Edge系列最高支援Wi-Fi 7，具備高達5.8 Gbps的增強速度、低於2毫秒
的低延遲率以及快速穩定的無線連線能力。Swift Edge 16更配備一系列智慧功能，提升
整體的生產力和易用性，使用者能利用1440p QHD網路攝影機進行視訊會議，更能透過
Window Studio Effects的自動取景、視線校正和進階背景模糊功能，以最佳的光線和角
度展現自己的專業能力；同時採用宏碁時序雜訊抑制（TNR）技術、Acer PurifiedVoice
AI 降噪技術，實現高品質的視訊通話體驗。Swift Edge 16亦配置一個全尺寸鍵盤和一系
列基本連接埠，包括兩個USB Type-A連接埠、具有快速充電和顯示功能的雙USB 4
Type-C PD 65 W 連接埠、HDMI 2.1和一個Micro SD讀卡機。

此外，宏碁發表Acer Connect Vero W6m網狀路由器，為旗下首款Wi-Fi 6E環保路由器，
機殼採用消費後回收（PCR）塑料製成，並具有高能源效率的節能模式。Acer Connect
Vero W6m系統搭載四核2 GHz處理器，配備一系列新升級的連線能力、覆蓋範圍和安全功
能，包括支援AXE7800[1,2] 三頻Wi-Fi 6E。
