: 原文內容：
: 石化產業在近五年，經歷全球景氣上升下跌、疫情後的爆發消費、烏俄戰爭衍生的能源價
: 格風波、台灣社會推展永續及政府淨零碳排的浪潮下等等，每一項都為石化產業的轉型帶
: 來挑戰，國喬石化身在其中，當然也無法例外。
: 5月初，國喬石化宣布今年通過普通股每股發放0.5元現金股利，當下引起市場關注，只因
: 業界都知道石化業這兩年有多困難，甚至允諾發放股利的國喬石化，在2022年每股仍虧損
: 0.56元。國喬對此表示，考量股東長期支持，故在虧損的情況下，仍決議發放股利，這樣
: 的決策代表國喬對於未來發展前景的信念與肯定。
: 國喬解釋，受市場波動影響，公司營收雖穩定，但中國同業的投產潮，對於以SM(苯乙烯)
: 作為主力產品的國喬，獲利空間持續受到壓縮。故自2018年起，國喬石化便大動作進行產
: 線調整、轉投資、甚至跨足新領域，為轉型開始鋪路，過去5年內也因此新增了166億新台
: 幣投資金額，正是受到股東們長期支持，國喬才得以在短期內完成布局，因此，即使在虧
: 損的2022年，國喬董事會仍通過0.5元的股利發放，以感謝股東長期支持。
: 5年前國喬與奇美實業合資，在古雷港經濟開發區設立漳州奇美，並於2021年9月正式量產
: ，上看60萬噸的ABS產能，還有今年投產的35萬噸PS產線，以及預計2025年開出18萬噸的P
: C產線，未來總產能可達百萬噸等級。再加上2008年國喬子公司鎮江國亨與鎮江奇美合併
: ，成為中國大陸最大的ABS製造商，因而國喬自2017年至2021年，5年內的獲利收入約有5
: 成來自於鎮江奇美及漳州奇美所認列之投資收益。
: 而國喬的企圖心不僅止於此，面對全球市場波動、地緣政治風險等壓力，要穩當的永續經
: 營需要更全面的布局。故在2019年，國喬獨資挑戰泉州PDH(丙烷脫氫)-PP(聚丙烯)一貫化
: 工廠，一個完全不同於SM的產品線，看重的不僅是風險分散，更是看好泉州石化園區擁有
: 上游原料來源及下游市場出處之完整配套措施，預計於2023年第四季即可商轉。
: 短短5年內，多項轉型投資支出，對於公司的財務壓力極其龐大，所幸前幾年趕上全球市
: 場上升期，超額的獲利挹注國喬順利推展轉型投資，眼看近乎就要迎來收成，國喬石化也
: 不吝與股東分享。在今年0.5元的現金股利後，漳州奇美ABS/PS/PC的投產，和泉州國亨66
: 萬噸丙烯、45萬噸聚丙烯，未來會給股東帶來更大的驚喜。
: 心得/評論：
: 國喬這幾年進行產線調整、轉投資，也跨新領域，成果頗豐，看好接下來國喬給股東的大
: 禮
